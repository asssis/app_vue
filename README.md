# app_vue

Teste feito para a empresa DonAdvice, teste de responsividade e HTML, CSS, Javascript e VUE

## Rodar Projeto

* instalar o docker
* instalar o docker-compose
* rodar o comando abaixo
```
 docker-compose up
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development

```
npm run lint
```
